Hi everyone, Following up on the previous post I shared last week — the Bulk UI Policy Actions Update Set is now available for anyone who wants to use it. What it solves: If you've built UI Policies for Catalog Items with many variables, you know the drill — open the policy, go to the related list, click New, fill every field, save, repeat. For 15 variables, that's 15 separate form submissions just to set Readonly/Mandatory/Visible on each one.
