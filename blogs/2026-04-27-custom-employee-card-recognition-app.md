<p>Virtual Cards is a custom ServiceNow application that lets employees send collectible recognition cards to colleagues. Not a Teams message that gets buried. Not a feed post nobody scrolls back to. An actual card — something you receive, keep, and collect.</p><p>&nbsp;</p><p>The idea came from CreatorCon collectables. The build came from pushing ServiceNow's vibe-coding capabilities as far as they'd go. The result is a scoped app built on reusable platform components — CMDB product models, Flow Designer, Service Portal widgets, a single Script Include as the logic hub — with an OpenAI integration for card image generation baked in.<br /><br /></p><p>Is it perfect? No. But it works, it's open source, and it's yours to use or build on.</p><p>&nbsp;</p>
