# ServiceNow

> ServiceNow provides a cloud-based Now Platform that automates enterprise IT, customer, employee, and operations workflows. Its inbound REST APIs expose CRUD access to any platform table plus dedicated interfaces for ITSM, CMDB, CSM, HRSD, ITOM, DevOps, and TM Forum Open APIs. All APIs are served from a customer instance at https://{instance}.service-now.com and are authenticated with Basic auth or OAuth 2.0.

This llms.txt was generated by API Evangelist from the curated ServiceNow catalog (apis.yml) and the artifacts in this repository. It is not published by ServiceNow.

## Core Platform APIs
- [Table API](https://www.servicenow.com/docs/bundle/yokohama-api-reference/page/integrate/inbound-rest/concept/c_TableAPI.html): CRUD on any ServiceNow table (openapi/servicenow-table-api-openapi.yml)
- [Aggregate API](https://www.servicenow.com/docs/bundle/yokohama-api-reference/page/integrate/inbound-rest/concept/c_AggregateAPI.html): count/avg/min/max/sum statistics (openapi/servicenow-aggregate-api-openapi.yml)
- [Attachment API](https://www.servicenow.com/docs/bundle/yokohama-api-reference/page/integrate/inbound-rest/concept/c_AttachmentAPI.html): file attachments on records (openapi/servicenow-attachment-api-openapi.yml)
- [Import Set API](https://www.servicenow.com/docs/bundle/yokohama-api-reference/page/integrate/inbound-rest/concept/c_ImportSetAPI.html): push external data through transform maps (openapi/servicenow-import-set-api-openapi.yml)
- [Batch API](https://www.servicenow.com/docs/bundle/yokohama-api-reference/page/integrate/inbound-rest/concept/batch-api.html): multiple REST requests in one call
- [Scripted REST APIs](https://www.servicenow.com/docs/bundle/yokohama-api-reference/page/integrate/custom-web-services/concept/c_CustomWebServices.html): custom server-side JavaScript endpoints
- [GraphQL API](https://docs.servicenow.com/bundle/tokyo-application-development/page/integrate/graphql/concept/scripted-graph-ql.html): client-driven queries (graphql/servicenow-graphql.md)
